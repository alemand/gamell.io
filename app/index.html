  • AdUpdater robot - Design and implement offline process to update prices in text Ads using Google Adwords API. Ads in Adwords can be tagged in order to be updated using Adwords API to have better ads that attract more customers and increase CTR.
  • ReportDownloader robot - Design an offline process to download Adwords reports using Google Adwords API. Clicks Performance, Keywords Performance and Campaign Performance are some of this reports.
  • TSOD robot - Design a solution to communicate customers, changes in their bookings using external API’s like Sound of Data to send sms messages and eCircle to send newsletters.
  • Feed robot - Design and implementation of xml feed creation for network affiliates like Criteo and TradeDoubler. Those feeds should show routes with minimum prices from searches done by users in our site to be shown in affiliates advertisements.
  • Identify, debug and fix live-site issues (Google Analytics, etc.)

All the above while embracing Agile development methodology, continuous deployments (Dev-Ops) and following the Clean Code ethos to improve code structure and architecture to make it more modular, scalable and maintainable.
